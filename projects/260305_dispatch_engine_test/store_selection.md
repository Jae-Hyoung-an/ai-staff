### 모니터링 지표

| 지표 | 측정 방법 | 기대 수준 |
|------|----------|----------|
| 배차 응답 시간 | 주문 접수 → 배차 완료 소요 시간 | < 30초 |
| 경로 최적화율 | 엔진 제안 경로 vs 실제 이동 거리 | 오차 < 20% |
| 조별 처리량 균형 | 조간 처리 건수 편차 | 편차 < 30% |
| 총 처리 건수 | 3시간 동안 완료된 총 배송 건수 | 36건 이상 |
| 배송 소요 시간 | 픽업 → 배달 완료 평균 시간 | < 20분/건 |

### 주의사항

- 지난주 목요일 데이터 기준이므로 실제 테스트일 주문량은 **+-20% 변동** 가능
- 12시대(점심)에 주문이 집중되는 패턴 → 11시에 워밍업, 12시에 본격 테스트
- Option B 선택 시 13시대 주문이 적어 후반부 테스트 밀도가 낮을 수 있음
- 테스트 상점 외 일반 상점 주문은 기존 기사가 처리하도록 사전 세팅 필요
