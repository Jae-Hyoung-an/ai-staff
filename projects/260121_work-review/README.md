## 왜 이 프로젝트가 필요한가?

### 현재 상황
- 연말평가 초안에서 **성과**는 잘 정리됨
- 하지만 **개선점**이 추상적 ("문서화 일관성", "데이터 분석 역량" 등)
- LV6 역량 기준과 현재 산출물 간 Gap 분석이 부재

### 기대 효과
1. 내 산출물의 **강점/약점**을 구체적으로 파악
2. **LV6 기준**에서 부족한 부분 명확화
3. **개선 액션 플랜** 수립

## 평가 대상 산출물

| 유형 | 예시 | 평가 관점 |
|------|------|----------|
| **PRD** | NVP 기사앱 PRD 50건 | 완성도, 명확성, 엣지케이스 |
| **데이터 분석** | 수락률-거리 관계 분석 | 인사이트 깊이, 액션 연결성 |
| **보고서** | 묶음배차 설문 분석 보고서 | 구조, 설득력, So What |
| **로드맵** | Road to AI 로드맵 | 전략적 사고, 실행 가능성 |
| **정책 설계** | 탐색배차 할증 정책 | 논리 구조, 트레이드오프 분석 |

## 평가 프레임워크

### LV6 역량 기준 (연말평가 기준)

| 역량 | LV6 기대 수준 | 평가 질문 |
|------|--------------|----------|
| 🧭 전략적 기획 | 전사 목표 기반 전략·로드맵 설계 | 산출물이 전사 목표와 연결되어 있는가? |
| 🔬 고급 분석 | 대규모 실험 설계·결과 분석 | 분석이 비즈니스 의사결정을 지원하는가? |
| 🏛️ PMO 거버넌스 | 전사적 거버넌스 체계 구축 | 표준화·재사용 가능한 프레임워크가 있는가? |
| 🔄 변화관리 | Transformation 주도 | 변화에 대한 저항을 어떻게 다루었는가? |
| 🤝 멘토링 | 교육·조직문화 리더십 | 팀원 성장에 기여한 산출물이 있는가? |

### 산출물 품질 체크리스트

#### PRD
- [ ] 문제 정의가 데이터로 뒷받침되는가?
- [ ] 성공 지표가 측정 가능하고 구체적인가?
- [ ] 엣지케이스가 충분히 고려되었는가?
- [ ] 개발자가 추가 질문 없이 작업 가능한가?
- [ ] 트레이드오프가 명시되어 있는가?

#### 보고서/분석
- [ ] 결론이 먼저 나오는가? (BLUF)
- [ ] 데이터 근거가 명확한가?
- [ ] So What이 있는가? (액션 아이템)
- [ ] 대상 독자에 맞는 깊이인가?

#### 로드맵/전략
- [ ] 전사 OKR과 연결되어 있는가?
- [ ] 우선순위 기준이 명확한가?
- [ ] 리소스/일정 현실성이 있는가?
- [ ] 리스크가 식별되어 있는가?
