### 프로덕트 영향범위

| 영역 | 주요 기능 |
|------|----------|
| **신규 앱** | 메인화면 배너, 지도/위치, 오더 제안, 수행 순서 안내, 상태 변경, 배차 취소 |
| **인트라** | 배너 등록/관리 |
| **매니저 PC/M** | NVP 기사 상태/위치 확인 (배차/상태변경 불가) |

> 권한 적용 메모: 어드민 계정이 `매니저PC`로 로그인한 경우에도 매니저 PC와 동일한 제약을 적용한다.  
> 단, 오더 상태값 기준 `대기(접수 계열)` 오더는 조회 가능하다.

### 주요 기능 (P0)

1. **메인화면 배너** - 앱 진입 시 팝업 배너, 슬라이드 형태
2. **지도 & 위치** - 기사 위치 표시, 확대/축소, 방향 표시
3. **신규 오더 제안 On/Off** - 제안 여부 선택, 자동 Off 정책
4. **오더 제안 받기** - 푸시, 지도 표시, 바텀시트 정보
5. **수행 순서 안내** - 다음 이동 지점만 표시, 목록 확인
6. **상태별 정보 표시** - 배정/도착/픽업 상태별 바텀시트
7. **상점 도착 처리** - 거리 기반 검증 (100m), bypass 기능
8. **음식 픽업 처리** - 위치 체크 없음, 로그 기록
9. **배송 완료 처리** - 사진 촬영, 후불결제, 위치 검증
10. **배차 취소** - 픽업 전/후 정책 구분, 횟수 제한, 사유 입력
11. **인트라 상태 변경 알림** - 수행 순서 갱신, 노티
12. **수행 중 오더 제안** - On/Off 스위치

### 특이사항

- **회의록 포함**: 문서 후반부에 리뷰 회의 받아쓰기 내용 포함 (약 400줄)
- **액션 아이템**: 6개의 미완료 액션 아이템 존재
- **정책 미확정**: 일부 정책값(N회 등)이 "논의 필요"로 표기
