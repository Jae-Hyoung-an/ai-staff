# _scratch 폴더 사용 가이드

> 임시 산출물/실험 파일을 모아두는 공간입니다. 정식 산출물은 프로젝트 폴더로 이동합니다.

---

## 분류 원칙

- 임시 문서/아이디어 메모: `_scratch` 유지
- 확정된 결과물: `projects/YYMMDD_프로젝트명/`으로 이동
- 이미지/HTML/스크립트 혼재 시: 파일명 접두어로 용도 구분
  - 문서: `YYYYMMDD_제목.md`
  - 목업: `YYYYMMDD_제목.html`
  - 스크립트: `YYYYMMDD_제목.py`
  - 이미지: `YYYYMMDD_제목.png`

---

## 정리 체크리스트

- [ ] 최근 2주 이상 미사용 임시 파일 확인
- [ ] 정식 프로젝트로 승격할 파일 이동
- [ ] 파일명 공백/특수문자 최소화
- [ ] 작업 종료 시 요약을 `knowledge/work_log/`에 기록
