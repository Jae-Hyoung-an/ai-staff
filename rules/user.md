## 반복 패턴 (자동 수집)

> AI가 발견한 반복 패턴이 여기에 기록됩니다.
> "user.md에 추가할까요?" 승인 후 자동 추가.

### 문서 작성

- **경영진/실무 이중 버전**: 주요 산출물은 경영진 요약본(~10장) + 실무 상세본(~20장) 두 벌로 작성하는 패턴
- **Mermaid 다이어그램**: SOP, 워크플로우, 오더 수행 플로우 등에서 적극 활용
- **표 중심 정책 정의**: 정책 항목을 항목-정책-비고 3열 표로 구조화

### 피드백 스타일

- (사용하면서 추가 예정)

### 워크플로우

- **HTML 파일 열기**: 항상 Chrome 브라우저로 열기
  - 명령어: `& "C:\Program Files\Google\Chrome\Application\chrome.exe" "{파일경로}"`
  - 파일명에 한글이 포함된 경우 영문으로 변경 후 열기

- **PDF/PPT 분석 요청 시**: document-importer 도구 활용
  1. 스크립트 실행: `python tools/document-importer/main.py` (inbox/ 자동 스캔)
  2. 이미지 분석하여 다이어그램, 표, 차트 내용 파악
  3. 텍스트 + 이미지 분석 결과 종합
  4. **페이지 개념 없이** 재구조화된 마크다운 생성
  5. **`analysis_result.md` 파일로 저장** → 임시 폴더에 저장하여 에디터에서 바로 확인 가능
  6. 사용자 리뷰 요청 → 수정사항 반영
  7. "확정" 시: knowledge/ 저장, resources/ 원본 이동, 임시 파일 삭제
