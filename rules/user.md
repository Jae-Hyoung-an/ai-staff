### 워크플로우

- **PDF/PPT 분석 요청 시**: document-importer 도구 활용
  1. 스크립트 실행: `python tools/document-importer/main.py` (inbox/ 자동 스캔)
  2. 이미지 분석하여 다이어그램, 표, 차트 내용 파악
  3. 텍스트 + 이미지 분석 결과 종합

  6. 사용자 리뷰 요청 → 수정사항 반영
  7. "확정" 시: knowledge/ 저장, resources/ 원본 이동, 임시 파일 삭제
