## 1. 벤더 등록기능

**개발도메인:** 인트라

| 항목 | 개요 | Priority | Detail |
|------|------|:--------:|--------|
| **벤더 등록/관리** | 사업자 정보 / 정산 정보 포함한 벤더 개념 등록 | P0 | **인트라 상 벤더 관리 메뉴 추가** - 목록 필터로 표시, 등록/삭제 가능 <br><br>**등록시:** <br>- 수아 필요 자료: 사업자 정보, 희망 지역(시군구 단위), 사업자 등록증, 계약서 이미지(업로드) 등 <br>- 사업자 등록시 중복 불가 정책 (지점 시 Validation) <br><br>**관리시:** <br>- 벤더 등록 정보 및 지점 연결 정보 수정 가능 <br>- 물량 수주 정책 |
| **벤더 상태 관리** | 벤더 상태를 대기 / 활성화 / 해지 상태로 조정 | P0 | **벤더 상세 페이지 진입시:** <br>- 벤더의 상태관리 기능(활성화/해지 상태로 변경 가능) <br>- 조직 이름 변경 기능 |
| **벤더-지점 연결** | 지점(권역)과 벤더 1:N으로 연결 기능 | P0 | **연결 기능:** <br>- 1개 타 지점 연결 가능 (1개 지점에 여러 벤더의 연결 검사/삭제) |
| **벤더 코드 발급** | 플러스앱 상 입력시 벤더장으로 자동 등록되는 코드 발급 | P0 | 타 해당 기사로 등록되기 위한 초대 코드 발급 - 추가 정보 코드 자동 생성 |
| **초대 코드 발급** | 플러스앱 상 입력해 벤더 기사로 입력되는 코드 발급 | P0 | 벤더 상세 입력해 초대 코드 발급 - 추가 발급 가능 |

## 2. 지점/권역관리

**개발도메인:** 인트라

| 항목 | 개요 | Priority | Detail |
|------|------|:--------:|--------|
| **지점타입 추가** | A타입 지점과 B타입 지점 구분간 추가 | P0 | **기존 지점 상세 페이지 외 정보 상 지점 operation type 의 NVP-B 타입 선택 추가.** <br><br>NVP 타입 지점 저장 시스템 시, 타 지점의 sales 권역과 해당 지점 세일즈 권역 등록 여부 체크. 중복시 저장 불가. |
| **권역 관리 설정 메뉴 신설** | NVP-B 지점 별 설정값 관리 목적 | P0 | 세트 단위물량 / 세트 단위 인센티브 지급 조건 및 금액 / 현황? 오금지 항 필요으로 관리. <br>설정값 관리 생성/표출/업(삭제/제)거 기능. |
| **설점간-권역 연결** | 설정값이 적용될 권역 지정 기능 | P0 | **연결 권역 탭 >** NVP-B 타 지점 검색 지점 기능. 타 여러가 지정 기능. <br>권역 상 설정값의 선택하여 상세 페이지 가능. <br>권역 별 물량 및 수 입력/저장 기능. |
| **세트 단위물량 관리** | 권역 별 세트의 단위물량 설정 기능 | P1 | 세트 기준 권역 별 > 최소 수행건수 (%) 입력 기능. |
