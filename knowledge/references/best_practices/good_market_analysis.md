# [Best Practice] 좋은 시장/경쟁 분석

> **핵심 메시지**: 시장 분석의 목적은 "시장이 얼마나 큰가"를 아는 것이 아니라,  
> **"우리가 어디서 싸워야 이길 수 있는가"**를 알아내는 것이다.

---

## 핵심 요약

시장/경쟁 분석은 전략적 의사결정의 기반이다. 단순히 시장 규모를 나열하는 것이 아니라, **우리 제품이 차별화할 수 있는 기회 영역**, **경쟁사 대비 강점/약점**, **시장의 변화 방향**을 파악하여 전략 수립에 활용해야 한다. 데이터만 모으는 것이 아니라 **인사이트를 도출**하는 것이 핵심이다.

## 1. 시장 분석 프레임워크

### 1.1 시장 규모 추정 (TAM-SAM-SOM)

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│     ┌─────────────────────────────────────┐            │
│     │              TAM                     │            │
│     │      (Total Addressable Market)     │            │
│     │         전체 시장 규모               │            │
│     │    예: 국내 푸드테크 시장 25조        │            │
│     │    ┌───────────────────────┐        │            │
│     │    │          SAM          │        │            │
│     │    │  (Serviceable AM)     │        │            │
│     │    │   유효 시장 규모       │        │            │
│     │    │ 예: 음식 배달 시장 15조│        │            │
│     │    │  ┌─────────────┐      │        │            │
│     │    │  │     SOM     │      │        │            │
│     │    │  │ (Serviceable│      │        │            │
│     │    │  │  Obtainable)│      │        │            │
│     │    │  │ 실제 목표    │      │        │            │
│     │    │  │ 예: 3조     │      │        │            │
│     │    │  └─────────────┘      │        │            │
│     │    └───────────────────────┘        │            │
│     └─────────────────────────────────────┘            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**계산 방법**

| 방식 | 설명 | 적합한 경우 |
|------|------|-----------|
| **Top-down** | 전체 시장에서 점유율 적용 | 기존 시장, 데이터 풍부 |
| **Bottom-up** | 고객 수 × 객단가 × 빈도 | 신규 시장, 검증 필요 |
| **Value Theory** | 해결하는 문제의 가치 기반 | 혁신적 제품 |

**Bottom-up 예시**

```
[음식 배달 서비스 SOM 추정]

1. 타겟 고객 수
   - 대도시 1-2인 가구: 500만 가구
   - 배달 이용 의향자 비율: 60%
   → 타겟: 300만 가구

2. 주문 빈도 및 객단가
   - 월 평균 주문: 4회
   - 평균 객단가: 25,000원

3. 시장 규모 계산
   - 월: 300만 × 4회 × 25,000원 = 3,000억원
   - 연: 3.6조원
```

## 3. SWOT 분석

### 3.1 SWOT 매트릭스

```
┌────────────────────────┬────────────────────────┐
│      Strengths         │      Weaknesses        │
│        (강점)          │        (약점)          │
├────────────────────────┼────────────────────────┤
│ • 낮은 수수료          │ • 낮은 브랜드 인지도   │
│ • 빠른 의사결정        │ • 제한된 마케팅 예산   │
│ • 기술력 (추천 알고리즘)│ • 음식점 수 부족       │
│ • 고객 충성도 높음      │ • 지역 커버리지 제한   │
├────────────────────────┼────────────────────────┤
│      Opportunities     │        Threats         │
│        (기회)          │        (위협)          │
├────────────────────────┼────────────────────────┤
│ • 1인 가구 증가        │ • 대형 플랫폼 공격적 투자│
│ • 배달 시장 성장       │ • 수수료 규제 논의      │
│ • 음식점 디지털 전환    │ • 라이더 인력 부족     │
│ • B2B 단체 배달 수요    │ • 경기 불황 우려       │
└────────────────────────┴────────────────────────┘
```

### 3.2 SWOT → 전략 도출

```
┌────────────────────────┬────────────────────────┐
│     SO 전략            │     WO 전략            │
│ (강점으로 기회 활용)    │ (약점 보완해 기회 활용) │
├────────────────────────┼────────────────────────┤
│ 낮은 수수료로          │ B2B 시장 진출로        │
│ 소형 음식점 공략       │ 브랜드 인지도 확보     │
├────────────────────────┼────────────────────────┤
│     ST 전략            │     WT 전략            │
│ (강점으로 위협 대응)    │ (약점/위협 최소화)     │
├────────────────────────┼────────────────────────┤
│ 기술력으로 효율화하여  │ 특정 지역 집중 전략    │
│ 비용 경쟁력 유지       │ (선택과 집중)          │
└────────────────────────┴────────────────────────┘
```

## 4. 경쟁 분석 실무

### 4.1 정보 수집 방법

| 소스 | 정보 유형 | 활용 방법 |
|------|----------|----------|
| **공개 정보** | 재무, 전략 방향 | 사업보고서, IR, 언론 |
| **제품 분석** | 기능, UX, 가격 | 직접 사용, 스크린샷 |
| **사용자 리뷰** | 만족/불만 포인트 | 앱스토어, 커뮤니티 |
| **업계 리포트** | 시장 규모, 트렌드 | 컨설팅사 보고서 |
| **인터뷰** | 심층 인사이트 | 전 직원, 파트너, 고객 |

### 4.2 경쟁사 모니터링 프로세스

```
[정기 모니터링 - 월간]

1. 제품 업데이트 추적
   - 앱 업데이트 내역
   - 새 기능 출시

2. 마케팅 활동
   - 프로모션, 광고
   - SNS 활동

3. 언론/발표
   - 보도자료
   - 컨퍼런스 발표

4. 사용자 반응
   - 앱스토어 리뷰 변화
   - 커뮤니티 반응
```

## 5. 시장 분석 보고서 작성

### 5.1 보고서 구조

```
1. Executive Summary (핵심 요약)
   - 시장 기회/위협 한 줄 정리
   - 핵심 인사이트 3가지
   - 전략적 시사점

2. 시장 개요
   - 시장 정의 및 범위
   - 시장 규모 (TAM/SAM/SOM)
   - 성장률 및 트렌드

3. 경쟁 분석
   - 주요 플레이어 현황
   - 경쟁 구도 (Five Forces)
   - 포지셔닝 분석

4. 기회 영역
   - 언더서브드 니즈
   - 경쟁 공백
   - 트렌드 기반 기회

5. 전략적 시사점
   - SWOT 기반 전략 방향
   - 우선순위 액션
```

### 5.2 인사이트 도출 예시

```
[데이터]
"쿠팡이츠 배달 시간 20분, 배민 35분"

[So What?]
"배달 속도에서 쿠팡이츠가 우위"

[Then What?]
"속도 경쟁은 인프라 투자 싸움.
우리는 속도 대신 '맛집 큐레이션'으로 차별화"

[Action]
"AI 추천 알고리즘 고도화 + 맛집 인플루언서 협업"
```

## 6. 트렌드 분석

### 6.1 PEST 분석

```
┌────────────────────────────────────────────────────────────┐
│                     PEST 분석                              │
├────────────────────────┬───────────────────────────────────┤
│   Political (정치)     │   Economic (경제)                 │
├────────────────────────┼───────────────────────────────────┤
│ • 플랫폼 규제 강화     │ • 경기 침체 우려                  │
│ • 배달료 상한제 논의   │ • 물가 상승 → 외식비 부담↑        │
│ • 라이더 고용 이슈     │ • 소비 양극화                     │
├────────────────────────┼───────────────────────────────────┤
│   Social (사회)        │   Technological (기술)            │
├────────────────────────┼───────────────────────────────────┤
│ • 1인 가구 증가        │ • AI/ML 추천 기술                 │
│ • 비대면 선호 지속     │ • 로봇/드론 배달                  │
│ • 건강식 관심 증가     │ • 슈퍼앱 트렌드                   │
└────────────────────────┴───────────────────────────────────┘
```

### 6.2 트렌드 → 기회 연결

| 트렌드 | 시사점 | 기회 |
|--------|--------|------|
| 1인 가구 증가 | 소량 주문 증가 | 1인분 전문 음식점 제휴 |
| 건강 관심 | 건강식 수요 | 영양 정보 제공, 건강식 카테고리 |
| 비대면 선호 | 언택트 서비스 | 비대면 결제, 문앞 배달 |
| AI 발전 | 개인화 기대 | AI 맛집 추천, 리오더 예측 |

## 7. 피해야 할 안티패턴

### ❌ 안티패턴 1: 숫자만 나열

```
나쁜 예:
"시장 규모 25조, 성장률 15%, 경쟁사 5개"
→ 그래서 뭘 해야 하는지 불명확

좋은 예:
"25조 시장에서 성장률 15%는 기회.
하지만 상위 3개사가 95% 점유.
틈새 전략 필요: B2B 또는 특정 지역 집중"
```

### ❌ 안티패턴 2: 경쟁사 과소/과대평가

```
과소평가:
"경쟁사 기술력은 별로다" → 근거 없는 낙관

과대평가:
"경쟁사가 너무 강해서 힘들다" → 포기 유도

올바른 접근:
객관적 데이터 기반 + 우리 강점 활용 전략
```

### ❌ 안티패턴 3: 일회성 분석

```
문제:
전략 수립 시에만 분석 → 이후 변화 놓침

해결:
• 정기적 모니터링 체계
• 경쟁사 업데이트 알림 설정
• 분기별 시장 분석 리뷰
```

## 8. 체크리스트

### 시장 분석 시
- [ ] TAM/SAM/SOM 구분이 명확한가?
- [ ] 시장 규모 추정 근거가 있는가?
- [ ] 성장 동인과 억제 요인을 파악했는가?
- [ ] "So What?"에 답하고 있는가?

### 경쟁 분석 시
- [ ] 직접/간접 경쟁사를 모두 포함했는가?
- [ ] 경쟁사의 강점/약점이 객관적인가?
- [ ] 우리의 차별화 포인트가 도출되었는가?
- [ ] 포지셔닝 기회가 명확한가?

### 보고서 작성 시
- [ ] 핵심 인사이트가 명확한가?
- [ ] 전략적 시사점이 있는가?
- [ ] 데이터 소스가 명시되어 있는가?
- [ ] 액션 아이템이 도출되었는가?
