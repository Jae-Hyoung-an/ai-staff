# 부릉 플러스+ (VROONG) 세일즈킷

> **핵심 슬로건**: "퀵보다 싸게, 택배보다 빠르게"

---

## 1. 핵심 가치 제안

### 지점 운영의 패러다임 전환: "관제" → "자동화"

> AI가 배차를 돌리는 동안, 지점장님은 더 중요한 업무에 집중하실 수 있습니다.

---

## 2. 주요 기능 및 혜택

### 2.1 관제/관리 리소스 절감

**기존 문제점 (Pain Points)**

| 문제 | 설명 |
|------|------|
| 안전 위험 | 배달하며 스마트폰으로 관제하는 것이 위험 |
| 관리 불가 | 한 번에 많은 콜이 오면 관리가 불가능 |
| 시간 부족 | 배달하느라 시간이 없어 영업 활동 불가 |

**부릉 플러스+ 솔루션**
- 기존 **수동 배차**에서 벗어나, **시스템이 자동으로** 기사에게 배차 처리
- 관리/관제 시간을 절약하여 **다른 업무(영업 등)에 집중** 가능

---

### 2.2 초보 기사도 바로 전력화

> "지점장님이 일일이 동선 가르치실 필요 없습니다."

| 항목 | 내용 |
|------|------|
| 교육 리소스 제로 | 지리 모르는 신입도 **AI 동선만 따라가면 OK** |
| 기사 수급/교육 고민 완화 | 초보 기사도 빠르게 적응 → 운영 안정화 |

**AI 배차 시 고려 정보**
- **기사 상황 정보**: 수행 중인 오더 상황 모두 반영
- **출/도착 정보**: 출/도착지, 경로 정보 반영

---

### 2.3 도입 후 기대 효과

| 효과 | 설명 |
|------|------|
| ⏱️ **배차 시간 단축** | 기사가 알아서 배정되어 더욱 신속한 배달 가능 |
| 🏪 **로컬 상점 영업** | 신속한 배달로 로컬 상점 만족도 향상, 더 많은 주문 처리 가능 |
| 🌧️ **기상 악화 시 자동제어** | AI가 배달 반경 자동 조절, 무리한 배차 제한으로 사고 리스크 감소 |

## 3. 수익성 개선: 시간당 회전율을 올리는 배차

> **핵심 공식**: 같은 시간에 더 많은 콜 → 더 많은 수익

### 3.1 AI 배차 최적화 기능

| 기능 | 설명 |
|------|------|
| 📦 **묶음 배송 최적화** | 현재 보유 콜과 어울리는 최적 콜을 AI가 **실시간 계산** |
| 🚗 **공차 거리 제로화** | 배달 완료 직후, **가장 가까운 다음 콜**을 미리 배정 |
| 🍳 **조리 대기 최소화** | 식당 조리 데이터 학습 → **픽업 타이밍에 맞춰 배차** |
