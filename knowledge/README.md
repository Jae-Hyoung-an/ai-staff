# Knowledge Base 구조

> PO 업무 흐름에 맞춘 지식 관리 체계

---

## 📁 폴더 구조 개요

```
knowledge/
├── 1_discovery/          # 발견 & 리서치 (업무 시작점)
├── 2_planning/           # 기획 & 설계 (핵심 산출물)
├── 3_strategy/           # 전략 & 목표 (상위 레벨)
├── 4_communication/      # 커뮤니케이션 자료 (선택: 필요 시 생성)
├── work_log/             # 업무 로그 (진행 기록/메모)
├── references/           # 참조자료 (Input)
└── domain/               # 도메인 지식 (부릉 특화)
```

---

## 🔍 1_discovery/ - 발견 & 리서치

업무의 시작점. 문제를 발견하고 데이터로 검증하는 단계.

| 하위 폴더 | 설명 | 예시 |
|----------|------|------|
| `user_research/` | 사용자 리서치 결과 | 인터뷰 정리, 설문 분석 |
| `market_research/` | 시장/경쟁사 분석 | 트렌드 리포트, 벤치마킹 |
| `data_analysis/` | 데이터 분석 결과 | 쿼리 결과, 인사이트 정리 |

---

## 📝 2_planning/ - 기획 & 설계

PO의 핵심 산출물. 기획서와 실험 설계 문서.

| 하위 폴더 | 설명 | 예시 |
|----------|------|------|
| `prd/` | PRD 문서 | 제품 요구사항 문서 |
| `feature/` | 기능 기획서 | 상세 기능 명세 |
| `experiment/` | 실험 설계 | A/B Test 설계, 인과추론 |

---

## 🎯 3_strategy/ - 전략 & 목표

상위 레벨의 전략적 문서. LV6 역량과 연결.

| 하위 폴더 | 설명 | 예시 |
|----------|------|------|
| `okr/` | OKR, KPI 관련 | 분기별 목표, 지표 정의 |
| `roadmap/` | 로드맵 | 서비스 단위 로드맵 |
| `business/` | 비즈니스 전략 | 세일즈킷, Go-to-Market |

---

## 💬 4_communication/ - 커뮤니케이션

이해관계자와의 소통 자료. AI 활용 핵심 영역.

| 하위 폴더 | 설명 | 예시 |
|----------|------|------|
| `reporting/` | 경영진 보고 | 주간/월간 보고, 의사결정 자료 |
| `stakeholder/` | 이해관계자 협업 | 타부서 협업 자료, 설득 자료 |
| `presentation/` | 발표자료 | 사내 발표, 외부 발표 |

---

## 🧾 work_log/ - 업무 로그

진행 중인 업무의 **일일 기록/메모/중간 산출물 링크**를 쌓는 공간.

- 목적: “왜 이렇게 결정했는지(Decision Log)”와 진행 맥락을 빠르게 복기
- 예시: 주간 회의 메모, 이슈 타임라인, 실험 진행 로그, PRD 작성 히스토리

---

## 📚 references/ - 참조자료

업무에 참고하는 자료. 직접 작성한 것이 아닌 Input.

| 하위 폴더 | 설명 | 예시 |
|----------|------|------|
| `best_practices/` | 베스트 프랙티스 가이드 | PO 업무 가이드라인 |
| `templates/` | 템플릿 모음 | PRD 템플릿, 보고서 양식 |
| `external/` | 외부 참고자료 | JD, 논문, 아티클 |

---

## 🏢 domain/ - 도메인 지식

부릉 서비스에 특화된 지식.

| 하위 폴더 | 설명 | 예시 |
|----------|------|------|
| `vroong/` | 부릉 서비스 관련 | 서비스 구조, 운영 정책 |
| `glossary/` | 용어 정의 | 도메인 용어집 |

---

## 📌 파일 네이밍 규칙

```
YYYYMMDD_제목.md
```

| 요소 | 설명 | 예시 |
|------|------|------|
| `YYYYMMDD` | 작성일 또는 관련 날짜 | `20260121` |
| `제목` | 한글/영어, 언더스코어 | `부릉_플러스_세일즈킷` |
