## 앱 개요

**NVP 부릉플러스앱**은 NVP(직고용) 기사 전용 배달 수행 앱이다.

- 클라이언트 식별자: `riderplus` (AGENT_NEW_APP)
- 대상 고용형태: `DIRECTLY_EMPLOYED`
- 지점 Operation Type: NVP

### 앱 화면 구조

```
[스플래시] → [로그인] → [컴플라이언스 인증] → [메인 화면]
                                                  ├─ 오더 제안 On/Off (AI 추천배차)
                                                  ├─ 오더 수행 (지도 + 수행 목록)
                                                  └─ [마이페이지]
                                                       ├─ 내 정보
                                                       ├─ 내 지갑 (M캐시)
                                                       ├─ 공지사항
                                                       ├─ 배달 내역
                                                       ├─ 설정
                                                       ├─ VCC 채팅 (카카오톡)
                                                       └─ VCC 전화 (1800-8255)
```

## 주요 비즈니스 흐름 요약

### 기사 온보딩 → 오더 수행까지

1. **로그인**: NVP 지점 소속 + DIRECTLY_EMPLOYED + 활성화 상태인 기사만 가능
2. **컴플라이언스 인증**: 본인인증 → 필수약관 동의 → 고용/산재보험 → 면허 → 안전모 → 범죄경력회보서
3. **오더 받기 On**: AI 추천배차 후보 등록 → 최대 동시 3건 수행
4. **오더 수행**: 제안(30초 타이머) → 수락 → 상점 도착 → 픽업 완료 → 배달 완료(사진/결제)
5. **정산**: 배달 완료 → M캐시 입금 → 내 지갑에서 출금 신청

### 핵심 숫자

| 항목 | 값 |
|------|-----|
| 오더 제안 타이머 | 30초 |
| 최대 동시 수행 | 3건 |
| 무응답 거절 → 자동 Off | 연속 3회 |
| 미수행 → 자동 Off | 2시간 |
| 배차 취소 일일 한도 | 3회 |
| Access Token 유효기간 | 1시간 |
| Refresh Token 유효기간 | 90일 |
| 비밀번호 만료 | 90일 |
| 1회 출금 최대 한도 | 200만 원 |
| 1일 출금 최대 횟수 | 4회 |
| 출금 수수료 | 300원 |
| 상점 도착 인정 범위 | 반경 100m |
