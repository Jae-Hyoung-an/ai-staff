## 1. 공지사항

기사가 마이페이지를 통해 공지사항 화면에 진입, 게시판 형태로 공지 내용을 확인.

### 1-1. 진입 조건

| 항목 | 정책 |
|------|------|
| 진입 경로 | 마이페이지 > 공지사항 / 메인 화면 배너 클릭 (공지 상세 직접 진입) |
| 접근 가능 대상 | 로그인된 기사 |

### 1-2. 공지사항 목록 화면 구성

| 항목 | 정책 |
|------|------|
| 정렬 기준 | 게시 시간 기준 최근 순(내림차순) |
| 공지 노출 기준 | 인트라에서 설정된 게시 시간/만료기간 내 active 상태인 공지 |
| New 뱃지 표시 조건 | 게시 시점으로부터 **48시간** 이내 AND 미열람 상태 |
| 안읽음 표시 조건 | 공지사항 상세 화면 미열람 상태 |
