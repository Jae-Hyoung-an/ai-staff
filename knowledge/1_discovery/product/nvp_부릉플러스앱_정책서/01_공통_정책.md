## 1. 앱 접근 권한

NVP 부릉플러스 앱이 기사 단말기의 OS 기능에 접근하기 위해 동의를 요청하는 절차, 안내 방식, 거부 시 처리 기준.

### 1-1. 접근 권한 항목 총괄

| # | 권한 | 필수 여부 | 권한 요청 시점 | 용도 | 미허용 시 영향 |
|---|------|-----------|----------------|------|----------------|
| 1 | 위치 정보 | **필수** | 앱 처음 실행 시 / 앱 재실행 기준 로그인 시 1회씩 | 주변 오더 추천, 배달 위치 확인 | **서비스 이용 불가** |
| 2 | 신체 활동 | 선택 | 오더 받기 ON 시 / 앱 재실행 기준 1회씩 | 오더 추천·배달 위치 정확도 향상 | 이동 상태 기반 추천 정확도 저하 |
| 3 | 알림 | 선택 | 로그인 시 1회씩 | 신규 오더, 배달 현황, 공지 알림 | 알림 수신 불가 |
| 4 | 카메라 | 선택 | 최초 사진 촬영 시도 시 | 배달 완료 사진 촬영 | 배달 완료 사진 촬영 불가 |

### 1-2. 위치 정보 (필수)

**Android 네이티브 다이얼로그:**

| 선택지 | OS 동작 | 앱 동작 |
|--------|---------|---------|
| 앱 사용 중에만 허용 | WHILE_IN_USE 승인 | 정상 진행 |
| 이번만 허용 | 일회성 승인, 앱 종료 시 해제 | 정상 진행, 다음 실행 시 재요청 |
| 허용 안함 | 거부 | `필수 권한을 허용해 주세요` 다이얼로그 → 단말기 설정 유도 |

**iOS 네이티브 다이얼로그:**

| 선택지 | OS 동작 | 앱 동작 |
|--------|---------|---------|
| 한 번 허용 | authorizedWhenInUse 승인 | 정상 진행 |
| 앱을 사용하는 동안 허용 | 일회성 승인, 앱 종료 시 해제 | 정상 진행, 다음 실행 시 재요청 |
| 허용 안함 | 거부 | `필수 권한을 허용해 주세요` 다이얼로그 → 단말기 설정 유도 |

### 1-3. 신체 활동 (선택) - Android

| 노출 차수 | 대상 | 노출 형태 | 허용 시 | 허용 안 함 시 |
|-----------|------|-----------|---------|---------------|
| 1차 | 전체 | 권한 다이얼로그 | 오더 받기 정상 진행 | 오더 받기 정상 진행 |
| 2차 | 1차 "허용 안 함" 선택자 | 권한 다이얼로그 | 설정 페이지로 이동 | 오더 받기 정상 진행 |
| 3차~ | 1차 "허용 안 함" 선택자 | 커스텀 다이얼로그 → 설정 이동 | 앱 시스템 설정으로 이동 | 오더 받기 정상 진행 |

### 1-4. 알림 (선택)

- 요청 시점: 스플래시 화면에서 요청, 앱 재실행 시마다 재안내 (미허용 상태인 경우)

| 알림 허용 상태 | 앱 동작 |
|----------------|---------|
| 허용 | 메인 화면 정상 진입 |
| 거부 | "알림을 켜주세요" 커스텀 다이얼로그 노출 |

- `설정 열기` tap → 단말기 앱 알림 설정 페이지로 이동
- `다음에 하기` tap → 알림 미허용 상태로 서비스 이용 가능

### 1-5. 카메라 (선택)

| 선택지 | OS 동작 | 앱 동작 |
|--------|---------|---------|
| 앱 사용 중에만 허용 | 권한 승인 | 사진 촬영 가능 |
| 이번만 허용 | 일회성 승인 | 다음 촬영 시도 시 재요청 |
| 허용 안함 | 거부 | "사진 촬영을 허용해주세요" 커스텀 다이얼로그 |

- `설정 열기` tap → 시스템 카메라 액세스 권한 설정 페이지로 이동
- `다음에 하기` tap → 사진 촬영 불가 상태 유지

## 3. 메인 화면 배너 바텀시트

메인 화면 진입 시 노출되는 바텀시트 구성 요소 및 노출 조건.

| 항목 | 정책 |
|------|------|
| 바텀시트 노출 조건 | 로그인 후 앱 실행 시 항상 노출 |
| 노출 배너(콘텐츠) 조건 | 인트라 - '표시' 상태의 배너 |
| 최대 노출 가능 배너 개수 | 제한 없음 |
| 배너 노출 순서 | 인트라 상 최신 등록 배너가 먼저 보이도록 내림차순 |
| 오늘 그만보기 tap 시 | 내일(00시 기준) 재노출 |
