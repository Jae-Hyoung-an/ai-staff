# [NVP] 설정 정책

> 원본: Notion - [NVP] 설정 정책서 (강수은, 리뷰중)
> 최종 업데이트: 2026-03-24

---

## 1. 설정 페이지

기사가 마이페이지를 통해 앱 설정 및 약관 확인.

### 1-1. 진입 조건

| 항목 | 정책 |
|------|------|
| 진입 경로 | 마이페이지 > 설정 |
| 접근 가능 대상 | 로그인된 기사 |

### 1-2. 설정 페이지 구성

| 섹션 | 하위 메뉴 | 설명 |
|------|-----------|------|
| 배달 설정 | 내비게이션 연결 | 부릉플러스 앱의 지도 내비게이션 앱 종류 설정 |
| 약관 및 정책 | 서비스 정보 | 개인정보 처리방침 / 위치서비스 이용약관 / 부릉플러스앱 약관 |
| | 오픈소스 라이센스 | - |
| 앱 관리 | 오류 정보 보내기 | - |
| | 앱 버전 | 현재 기기에서 구동 중인 앱 버전 노출 |

---

## 2. 내비게이션 연결

오더 수행 중 경로 안내 시 연결할 내비게이션 앱 설정.

### 2-1. 진입 경로

| # | 경로 |
|---|------|
| 1 | 오더 수행 > 길찾기 tap 시 |
| 2 | 배달 설정 > 내비게이션 연결 tap |

### 2-2. 선택 가능한 내비게이션 앱

| # | 앱 |
|---|-----|
| 1 | 네이버지도 |
| 2 | 카카오맵 |
| 3 | 카카오내비 |
| 4 | 티맵 |

### 2-3. 주의사항

- 설치된 서비스만 선택 가능
- 미설치 내비게이션의 '설치' 버튼 → 스토어 링크 이동

---

## 3. 서비스 정보

기사가 약관·정책 섹션에서 서비스 관련 정보와 약관 내용 확인.

| 항목 | 설명 |
|------|------|
| 서비스 정보 | 개인정보 처리방침 / 위치서비스 이용약관 / 부릉플러스앱 약관 |
| 오픈소스 라이센스 | 앱에 사용된 오픈소스 라이센스 확인 |

---

## 4. 오류 정보 보내기

기사가 오류 정보를 보내 CS 해결 및 앱 품질 개선에 기여.

### 4-1. 수집 정보

- 계정 정보 (로그인 ID)
- 기기 정보 (OS 버전, 휴대폰 모델명)
- 오류 정보

### 4-2. 사용 시나리오

1. 기사가 VCC에 오류 접수
2. VCC가 오류 정보 보내기 안내, 개발팀에 문의
3. 개발팀에서 로그 확인 (RUM 로그가 담긴 sqlite DB 파일을 S3에 업로드 → sqlite browser로 확인)
4. 오류 원인 파악 및 해결
