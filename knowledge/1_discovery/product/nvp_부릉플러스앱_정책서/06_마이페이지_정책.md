## 1. 마이페이지

기사의 개인 정보, M캐시 잔고, 공지사항, 배달 내역, 앱 설정 등을 통합 조회·관리.

### 1-1. 화면 구성

```
[마이페이지]
  ├─ 기사 프로필 영역 (활동명, 기사 코드) → [내 정보]
  ├─ M캐시 보유액 표시
  ├─ 내 지갑 보기 → [내 지갑]
  ├─ 공지사항 → [공지사항]
  ├─ 배달 내역 → [배달 내역]
  ├─ 설정 → [설정]
  ├─ VCC 채팅 → [카카오톡 상담채널]
  └─ VCC 전화 → [네이티브 통화 앱 (1800-8255)]
```

### 1-2. 공지사항 'N' 뱃지

- 노출 조건: 공지사항 게시 시점으로부터 **48시간** 이내 AND 미열람 상태

### 1-3. 진입 경로 및 조건

| 항목 | 정책 |
|------|------|
| 진입 경로 | 메인 화면 상단 마이페이지 버튼 tap |
| 진입 조건 | 로그인 완료 상태에서만 접근 가능 / 출근·퇴근 상태 무관 |
