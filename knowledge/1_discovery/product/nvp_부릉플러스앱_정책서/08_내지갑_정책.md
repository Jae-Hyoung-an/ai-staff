# [NVP] 내 지갑 정책

> 원본: Notion - [NVP] 내 지갑 정책서 (강수은, 리뷰중)
> 최종 업데이트: 2026-03-24

---

## 1. 내 지갑

기사가 보유 M캐시 잔액·입출금 내역을 확인하고 출금 신청 또는 계좌 정보를 관리.

### 1-1. 진입 조건

| 항목 | 내용 |
|------|------|
| 진입 경로 | 마이페이지 > 내 지갑 보기 |
| 접근 가능 대상 | 로그인된 기사 |

### 1-2. M캐시 입금 항목

| 거래 종류 | 설명 | 비고 |
|-----------|------|------|
| 배송비 | 배달 완료 시 지급 (배달비 - 원천징수세) | - |
| 미션 달성 보상 | 미션 달성 시 즉시 지급 | 어떤 미션이 있는지는 P1 |
| 길찾기 꿀팁 선물 | - | - |

### 1-3. M캐시 출금 항목

| 거래 종류 | 설명 |
|-----------|------|
| M캐시 출금 | 기사 직접 출금 신청 금액 |
| 시간제 보험 사용료 | 보험 이용에 따른 차감 |

### 1-4. 출금 신청 버튼 활성화 조건

| 조건 | 미충족 시 처리 |
|------|----------------|
| 본인 인증 완료 | 구 기사앱에서 완료할 수 있도록 버튼 노출 |
| 주소 등록 완료 | 구 기사앱에서 완료할 수 있도록 버튼 노출 |
| 출금 실계좌번호 등록 완료 | 계좌정보 등록 페이지로 유도 툴팁 |
| 출금신청 권한 존재 | 부릉케어센터 전화 문의 버튼 |

---

## 2. 계좌 정보

기사가 본인의 M캐시 출금 계좌를 조회·등록·수정.

### 2-1. 예금주 입력 조건

| 항목 | 내용 |
|------|------|
| 허용 문자 | 한글, 영문(대/소문자), 숫자, 공백, 괄호 |
| 제한 문자 | `, . / \ @ # ! % ^ & *` 등 일반 특수문자, 이모지 및 기타 기호 |
| 기타 조건 | 첫 글자 공백 불가 / 연속 공백은 1칸으로 정규화 / 붙여넣기 불가 |

### 2-2. 실계좌 번호 입력 조건

| 항목 | 내용 |
|------|------|
| 허용 문자 | 숫자 |
| 제한 문자 | 숫자 외 모든 문자 |
| 기타 조건 | 붙여넣기 불가 |

---

## 3. 출금 신청

등록된 계좌로 M캐시 출금을 신청. 출금 금액 입력 → 비밀번호 확인 → 출금 처리.

### 3-1. 출금 가능 금액 계산

| 항목 | 정책 | 비고 |
|------|------|------|
| 출금 가능 금액 | 보유 M캐시 - 출금 불가 금액 | 실시간 계산 불가 (계좌 입금액, 시간제 보험 보증금, 수행보증금 제외) / 마이너스 시 '0원' 표시 |
| 출금 수수료 | **300원** | **[미확인]** 항상 300원인지 확인 필요 |
| 1회 출금 최대 한도 | **200만 원** | - |

### 3-2. 출금 불가 에러

| # | 항목 | 비고 |
|---|------|------|
| 1 | 비밀번호 불일치 | 비밀번호 확인 단계 |
| 2 | 회당 출금 가능 금액 초과 | 1회 최대 200만 원 |
| 3 | 1일 출금 가능 횟수 초과 | 1일 최대 **4회** |
| 4 | 24시간 이내 출금 신청 | - |
| 5 | 출금 불허용 | - |
| 6 | 계좌번호 부적합 | - |
| 7 | 미납금 존재 | - |
| 8 | 시스템 점검 중 | - |
| 9 | M캐시 잔액 부족 | - |
| 10 | 수수료 부족 | - |
