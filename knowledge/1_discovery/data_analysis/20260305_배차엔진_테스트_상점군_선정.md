## 핵심 발견

배차엔진 테스트 상점군 선정 시 두 가지 접근법이 있으며, 테스트 목적에 따라 선택해야 한다.

## 두 가지 Option 비교

| 항목 | Option A: 소수 고밀도 | Option B: 분산 |
|------|----------------------|----------------|
| 상점 수 | 5개 | 28개 |
| 컨셉 | 주문량 상위 상점 집중 | 실제 운영 환경 재현 |
| 테스트 포인트 | 대량 주문 동시 배분 성능 | 다양한 거리·밀도 조합 대응력 |
| 예상 물량 | 2시간 기준 ~35건 | 2시간 기준 ~52건 |

## 테스트 설계 전제

| 항목 | 값 |
|------|-----|
| 테스트 시간 | 목요일 11:00~14:00 (3시간) |
| 투입 조 | 3개 조 |
| 조당 처리량 | 4~6건/시간 |
| 총 처리 가능량 | 36~54건 |
| 배송 반경 | ~1.5~2km |

## 데이터 분석 결과

- 전체 주문 97건 중 상봉동 46%, 면목동 27% 집중
- 시간대별: 12시(42%) > 11시(32%) > 13시(26%)

## So What - 재사용 가이드

배차엔진 테스트 설계 시:
1. 대상 지점의 최근 실적 데이터에서 피크 시간대 주문 분포 추출
2. 고유 상점/위치 수, 동별 분포, 시간대별 분포 분석
3. 테스트 목적에 따라 고밀도(스트레스 테스트) vs 분산(현실성 테스트) 선택
4. 조별 초기 위치와 상점 간 거리 행렬을 사전 설계
